# 📊 Customer Churn Analysis

## 📌 Objective

This project analyzes customer churn data to identify key factors that lead to customer attrition and provide actionable business recommendations.

---

## 📊 Dataset

Telco Customer Churn Dataset
Kaggle: https://www.kaggle.com/datasets/blastchar/telco-customer-churn?resource=download

---

## 🛠 Tools & Technologies

* Python (pandas, numpy)
* Data Visualization (seaborn, matplotlib)

---

## 🔍 Analysis Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature relationship analysis
* Visualization of churn patterns

---

## 📈 Key Insights

* Customers on month-to-month contracts have the highest churn rate
* Customers with higher monthly charges are more likely to churn
* Customers with shorter tenure are at higher risk

---

## 💡 Business Recommendations

* Offer incentives for long-term contracts
* Provide discounts for high-paying customers
* Improve onboarding for new customers
